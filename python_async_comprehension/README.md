# Python - Async Comprehension

## Description
This project focuses on asynchronous generators and comprehensions in Python.

## Learning Objectives
* How to write an asynchronous generator
* How to use async comprehensions
* How to type-annotate generators

## Tasks
| Task | File | Description |
| ---- | ---- | ----------- |
| 0. Async Generator | 0-async_generator.py | A coroutine that yields random numbers after an async wait. |
| 1. Async Comprehensions | 1-async_comprehension.py | Collect 10 random numbers using async comprehension. |
| 2. Run time for four parallel comprehensions | 2-measure_runtime.py | Measure the runtime of parallel async comprehensions. |
